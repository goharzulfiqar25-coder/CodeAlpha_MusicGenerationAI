# 🎵 CodeAlpha Music Generation AI

An Artificial Intelligence based Music Generation System developed as part of the CodeAlpha Artificial Intelligence Internship Program.

## 📌 Project Description

This project uses Long Short-Term Memory (LSTM) Neural Networks to analyze MIDI music files, learn musical patterns, and generate new AI-composed music automatically. The system is capable of training on existing music datasets and producing unique musical sequences in MIDI format.

## 🚀 Features

- 🎼 MIDI File Processing
- 🤖 Deep Learning Model using LSTM
- 🎵 AI-Based Music Composition
- 🖥️ User-Friendly Tkinter GUI
- 💾 Export Generated Music as MIDI Files
- ⚡ Automatic Music Generation
- 📊 Music Pattern Learning and Prediction

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras
- Music21
- NumPy
- Tkinter
- MIDI Dataset

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/goharzulfiqar25-coder/CodeAlpha_MusicGenerationAI.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ How to Run

### Step 1: Preprocess Dataset

```bash
python preprocess.py
```

### Step 2: Train the Model

```bash
python train_model.py
```

### Step 3: Launch GUI

```bash
python gui.py
```

### Step 4: Generate Music

Click the **Generate Music** button and the generated MIDI file will be saved automatically.
